# Color-Palette
Recreate the CSS Grid Color Palette from Figma

This project demonstrates advanced layout structuring using CSS Grid and the Roboto font from Google Fonts.

It contains two main grid sections with nested grid layouts and a footer positioned using a full-page grid structure.
